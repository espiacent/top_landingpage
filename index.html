HTML isn't actually a coding language but a markup language that serves as the backbone of a website.

CSS is what gives all the elements defined in the HTML structure a layout and what makes (theoretically) a site pretty.

Javascript makes a site more responsive than can be achieved with with CSS alone. It is a coding language.

Python is a coding language that is a good starting point as its code is very readable.
